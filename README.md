# StudyFlow – Student Study Planner App

![Cover](images/cover.png)

## Overview
A student study planner app that tackles academic overwhelm through smart task management, a built-in Pomodoro timer, and streak tracking — designed around how students actually think and work.

## My Role
UI/UX Designer — research, wireframing, high-fidelity design, prototyping

## Process
1. Problem Discovery
2. User Flows
3. Wireframes
4. Visual Design (purple-based design system)
5. Prototyping

## Design System
- **Style:** Purple-based design system with a soft, modern aesthetic
- **Logo:** 4 SVG variants
- Consistent typography and component system across all screens
